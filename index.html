LLMuse Privacy Policy

BUILD · SHIP · SCALE
Effective date: February 24, 2025

TL;DR — LLMuse does not collect, store, or share any personal data. Everything stays on your device.

1. Data We Collect

None. LLMuse does not collect any personal information, analytics, telemetry, or usage data. We have no servers, no databases, and no tracking of any kind.

2. API Keys

Your API keys (Anthropic, OpenAI, GitHub) are stored exclusively in your device's Keychain, Apple's secure on-device credential storage. They are never transmitted to any server other than the official API endpoints of the respective providers (api.anthropic.com, api.openai.com, api.github.com).

3. Network Requests

LLMuse makes HTTPS requests solely to:

  • Anthropic API — to retrieve your Claude usage quotas
  • OpenAI API — to retrieve your Codex usage quotas
  • GitHub API — to retrieve your Copilot usage quotas

These requests use your own API keys and are made directly from your device. No data passes through any intermediary server.

4. On-Device Storage

LLMuse stores preferences (display settings, refresh intervals, notification thresholds) in local UserDefaults within an App Group shared between the app and its widget extension. This data never leaves your device.

5. In-App Purchases

LLMuse offers an optional one-time "Pro" upgrade processed entirely through Apple's StoreKit. We do not have access to your payment information. Apple handles all transactions.

6. Third-Party Services

LLMuse uses zero third-party SDKs, analytics tools, crash reporters, or advertising frameworks. The app has zero external dependencies.

7. Children's Privacy

LLMuse does not collect data from anyone, including children under 13.

8. Changes to This Policy

If we update this policy, the changes will be reflected on this page with an updated effective date.

9. Contact

Questions or concerns? Open an issue on GitHub.
